Upload area

For adding data to the DB, follow these steps:

  1. Important: use only the template CSVs as shown in the query tab. Swapping column positions or leaving selected fields blank is not a problem, but don't remove nor rename any column names.
  2. First, upload the general information regarding the measurement points (option 'Stations') using the appropriate template CSV. Name the "measurement stations" carefully. They will link the database across multiple tables to gather parametrical data for that measurement point. Thus each field of the columns "meas_station" in the CSV template needs to be unique.
  3. Upload the collected data (following the corresponding template) by selecting the data type and uploading the csv in the dropzone below. As long as the measurement stations (step 2) are uploaded to the database, the order for uploading the remaining parametrical data doensn't matter. Important: the measurement station of the parametrical data needs to exactly match one of the "meas_station" you provided in step 2, otherwise an error message will be shown.
Alternatively, the DB can also be managed in the Admin module.
